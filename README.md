# CNN-MNIST
基于CNN的MNIST手写数字识别

见代码注释

用法如下：

------

- [ ] 训练模型

命令:

python3 mnist.py train

- [ ] 测试集对模型进行测试

命令：

python3 mnist.py test

- [ ] 拿10个测试集样本进行预测

命令：

python3 mnist.py predict
